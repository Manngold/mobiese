# Welcome to Mobiese 🍿

## This is simple movie app using vanilla javascript

무한으로 즐겨요 인기 영화 목록 조회 사이트

infinity scroll을 통한 인기 영화 리스트 조회 앱

TMDB를 통해 인기 영화 리스트를 조회하고 있습니다.

### For what

- 함수를 값으로 다루기
- 이터러블/이터레이터 프로토콜
- 재사용성 높이기
- 로직의 가독성 높이기
- Promise에 대해 깨우치기
- 프론트엔드에서 함수형 프로그래밍 적용하기
